▶ How to connect your Google Sheet
1. Open your Google Sheet → File → Share → Publish to web
2. For the roster: select Roster sheet tab → CSV → Publish → paste URL above
3. For rules: select Rules sheet tab → CSV → Publish → paste URL above
4. Tap Sync Now — both sheets sync together
